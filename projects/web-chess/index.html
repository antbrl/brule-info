    Web-chess

    Jeu d'échecs en ligne fait avec Rust et VueJS
    juin 2021

    Démo - https://tranquil-savannah-84310.herokuapp.com/

    Ce projet a pour but d’explorer les capacités du langage Rust pour un serveur de jeu en ligne.

    Dans cette implémentation du jeu d’échecs, une partie a la particularité d’être définie uniquement par son historique d’actions – son état courant doit être recalculé à chaque étape.

    Code source - https://github.com/pop123123123/web-chess
